# Jadoluz Theme

A light theme for VSCode inspired by [Armadillo](https://github.com/rahmanyerli/armadillo).

## Development

### Requirements

#### npm

- `npm-run-all`: `npm install --global npm-run-all`
- `vsce`: `npm install --global vsce`

### Editing

- Fork & Clone this repository
- `npm install`

### Packaging

- `npm run all`

### Publishing

- `npm run publish`
